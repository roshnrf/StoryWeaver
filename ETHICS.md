# Ethics & Privacy Policy

## Data Protection Policy

This project **prioritizes child safety and data privacy**.

### What We Do NOT Store

❌ Raw audio recordings of children  
❌ Personal identifying information (names, ages, locations)  
❌ Video recordings  
❌ Unencrypted sensitive data  

### What We Store Locally

✅ Session metrics (accuracy, errors, timestamps)  
✅ Anonymized transcripts  
✅ Progress statistics  
✅ Configuration settings  

---

## For Researchers & Users

### Before Using This System

1. **Obtain Written Consent**
   - Get explicit parental/guardian permission
   - Explain data collection and usage
   - Provide opt-out options

2. **Anonymize Data**
   - Remove metadata from audio files
   - Use random IDs instead of names
   - Strip location information

3. **Secure Storage**
   - Store audio offline only
   - Use encrypted storage for sensitive data
   - Never upload to public cloud without consent

4. **Follow Regulations**
   - Comply with local IRB/ethics boards
   - Follow GDPR/COPPA guidelines
   - Respect children's rights

## Recommended Practices

### Audio File Management
```bash
# Good practice: Anonymized filenames
data/audio/child_001_session_01.wav
data/audio/child_002_session_01.wav

# Bad practice: Identifiable information
data/audio/john_doe_age6_recording.wav
```

### Consent Documentation

Keep offline records of:
- Signed consent forms
- Date of consent
- Scope of data usage
- Parent contact information
- Withdrawal procedure

---

## Limitations

⚠️ **This tool is NOT a replacement for professional speech therapy**

- Use as a supplementary practice tool
- Consult licensed speech-language pathologists
- Monitor child's progress with professionals
- Discontinue if child shows discomfort
